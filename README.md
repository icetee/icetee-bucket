# icetee-bucket
My custom manifests for Scoop.

## Usage

Dead simple! Install [Scoop](https://github.com/lukesampson/scoop) and run the
command:  
```
scoop bucket add icetee-bucket https://github.com/icetee/icetee-bucket
```
